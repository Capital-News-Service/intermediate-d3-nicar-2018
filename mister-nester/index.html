Hello, I’m Mister Nester

I’m a simple console for learning and experimenting with d3.js data nesting.

Forked by John Muyskens from the original by Shan Carter for the NICAR 2018 intermediate d3 workshop.

Data by David Eads for ProPublica Illinois for the story “Why (Almost) No One Is Charged With Gun Trafficking in Illinois” published Oct. 13, 2017 (link to analysis repository on GitHub).

Documentation links: d3.nest, statistics.

Uses the renderjson library by David Caldwell.
